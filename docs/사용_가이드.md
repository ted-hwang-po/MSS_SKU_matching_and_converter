# SKU 변환기 사용 가이드

## 시작하기

### 실행 방법
1. `SKU_변환기.exe` 파일을 더블클릭합니다.
2. Windows Defender SmartScreen 경고가 뜨면 → "추가 정보" → "실행" 클릭
   - 이 경고는 Microsoft 인증서가 없는 모든 새 프로그램에 표시되며, 보안 문제가 아닙니다.
3. 프로그램 창이 열리면 준비 완료입니다.

> 첫 실행 시 약 5~10초 정도 로딩 시간이 걸릴 수 있습니다.

### 새로운 기능 (v2)
- **프로그레스 바**: 처리 진행 상태를 시각적으로 확인
- **프리셋**: 자주 쓰는 파일+브랜드 조합을 저장/불러오기
- **세션 기억**: 마지막 사용한 파일/폴더를 앱 재시작 시 자동 복원
- **파일 검증**: 파일 선택 직후 필수 컬럼 존재 여부를 자동 확인
- **작업 이력**: 과거 변환 기록을 "작업 이력" 탭에서 조회
- **매칭 실패 리포트**: 경고 사항을 Excel로 내보내기
- **덮어쓰기 보호**: 기존 출력 파일이 있으면 확인 후 진행
- **취소 기능**: 처리 중 "취소" 버튼으로 중단 가능
- **드래그앤드롭**: 파일 탐색기에서 파일을 끌어다 놓기 지원 (tkinterdnd2 설치 시)
- **툴팁**: 각 입력 필드에 마우스를 올리면 설명 표시
- **초기화**: "초기화" 버튼으로 모든 입력을 리셋

## 기능 A: SKU 매칭 변환

### Step 1. 프리셋 (선택)

이전에 저장한 파일+브랜드 조합이 있으면 프리셋을 선택하고 "불러오기"를 클릭하세요.
처음 사용하는 경우 건너뛰고 Step 2로 이동합니다.

- **프리셋 저장**: 파일과 브랜드를 설정한 뒤 "저장" 버튼 클릭 → 이름 입력
- **프리셋 삭제**: 프리셋 선택 후 "삭제" 버튼 클릭

### Step 2. 입력 파일 선택

3개의 파일을 각각 "파일 선택" 버튼을 눌러 지정합니다.

| 순서 | 파일 | 설명 |
|------|------|------|
| 1 | **총 발주 수량 데이터셋** | 브랜드별 전체 발주 정보가 담긴 파일 |
| 2 | **상품코드-바코드 매칭 데이터셋** | 바코드와 상품코드(UID)를 매핑하는 파일 |
| 3 | **상품별 옵션 정보** | 옵션이 있는 상품의 사이즈유형/옵션 목록 파일 |

- 지원 형식: `.xlsx`, `.csv`
- 파일 1을 선택하면 자동으로 브랜드 목록이 체크박스에 채워집니다.
- **파일 검증**: 파일 선택 직후 `[OK]` 또는 `[!!]` 상태가 표시됩니다.
  - `[OK] 1234행, 5개 브랜드` — 정상 로드됨
  - `[!!] 필수 컬럼 누락: 바코드` — 파일이 올바르지 않음

> **드래그앤드롭**: tkinterdnd2가 설치되어 있으면 파일을 직접 끌어다 놓을 수 있습니다.

### Step 3. 브랜드 선택

두 가지 모드 중 하나를 선택합니다:

| 모드 | 설명 |
|------|------|
| **모든 브랜드** | 파일 1에 있는 모든 브랜드에 대해 일괄 변환 |
| **브랜드 선택** | 체크박스 목록에서 원하는 브랜드를 선택 |

- **검색**: "브랜드 선택" 모드에서 검색창에 키워드를 입력하면 목록이 필터링됩니다.
- **전체 선택/해제**: 현재 검색 결과에 보이는 브랜드만 선택/해제합니다.
- 선택된 브랜드는 상단에 "선택됨 (N개): 브랜드A, 브랜드B" 형태로 표시됩니다.

> "모든 브랜드"를 선택하면 브랜드별로 각각 2개씩 파일이 생성됩니다. (예: 브랜드 5개 → 총 10개 파일)

### Step 4. 저장 위치 선택

- 기본값은 바탕화면입니다.
- 다른 위치를 원하면 "폴더 선택" 버튼으로 변경합니다.

### Step 5. 변환 실행

- "변환 실행" 버튼을 클릭합니다.
- **미리보기 확인**: 처리할 브랜드 수, 생성될 파일 목록, 덮어쓰기 경고가 표시됩니다.
  - 기존 파일이 존재하면 경고가 함께 표시됩니다.
  - "확인"을 눌러야 실행이 시작됩니다.
- **프로그레스 바**: 현재 처리 중인 브랜드와 진행률이 표시됩니다.
  - 예: `브랜드A (3/10)` — 10개 중 3번째 처리 중
- **취소**: 처리 중 "취소" 버튼을 누르면 현재 브랜드 완료 후 중단됩니다.
- 완료되면 "완료" 메시지가 나타납니다.

### Step 6. 결과 확인

- "출력 폴더 열기" 버튼으로 저장 폴더를 열 수 있습니다.
- 생성되는 파일 2개:

| 파일 | 용도 |
|------|------|
| `{브랜드명}_시스템업로드_최종파일.xlsx` | 무신사 시스템 업로드용 |
| `{브랜드명}_발주리스트_최종파일.xlsx` | 브랜드 송부용 발주 리스트 |

### Step 7. 매칭 실패 리포트 (선택)

경고가 발생한 경우 "매칭 실패 리포트" 버튼이 활성화됩니다.
- 클릭하면 경고 상세 내용(브랜드, 88코드, 상품명, 원인)이 담긴 Excel 파일을 저장할 수 있습니다.
- 이 리포트를 참고하여 수동으로 확인/수정이 필요한 항목을 처리하세요.

## 입력 파일 필수 컬럼

> 아래 **필수 컬럼**이 입력 파일에 반드시 있어야 합니다. 컬럼명이 다르거나 누락되면 오류가 발생합니다.
> 그 외 컬럼은 있어도 무시되므로 삭제하지 않아도 됩니다.

### 파일 1 — 총 발주 수량 데이터셋

| 컬럼명 | 설명 | 예시 |
|--------|------|------|
| `브랜드명` | 브랜드 필터링 기준 | 자빈드서울 |
| `88코드` | 바코드 (13자리) | 8809298738242 |
| `상품명` | 상품 이름 | 윙크 파운데이션 팩트 15g 18호 |
| `공급가 (VAT 제외)` | 공급 단가 | 11455 |
| `정상가 (VAT 포함)` | 정상 판매가 | 28000 |
| `상시행사가 (VAT 포함)` | 상시 행사가 | 17000 |
| `수량(오프)` | 오프라인 발주 수량 | 100 |

- 가격 컬럼명에 줄바꿈이나 공백 차이가 있어도 자동 인식됩니다.
- `발주수량(Total)`이 아닌 **`수량(오프)`** 컬럼을 사용합니다.

### 파일 2 — 상품코드-바코드 매칭 데이터셋

| 컬럼명 | 설명 | 예시 |
|--------|------|------|
| `바코드` | 88코드와 동일한 값 | 8809298738242 |
| `1P 상품코드` 또는 `상품코드` | 상품코드 (UID) | 6199296 |
| `스타일번호` | 스타일 식별자 | JD_B_NWO |
| `옵션명` | 옵션 이름 (옵션 상품만) | 18 커버 라이트 |
| `상품명` | 기본 상품명 (옵션 제외) | 윙크 파운데이션 팩트 15g |

- 첫 번째 행이 비어있어도 자동 처리됩니다.
- `옵션명`과 `상품명`은 옵션 매핑 정확도를 높이므로 가급적 포함해주세요.

### 파일 3 — 상품별 옵션 정보

| 컬럼명 | 설명 | 예시 |
|--------|------|------|
| `사이즈유형` | 사이즈유형 코드 | SQ3 |
| `사이즈유형명` | 상품명 기반 매칭용 텍스트 | 자빈드서울 윙크 파운데이션 팩트 15g |
| `Size01` ~ `Size30` | 옵션 값 목록 | 18 커버 라이트 |

- Size 컬럼에 모든 옵션이 등록되지 않아도 파일 2의 순서를 기반으로 자동 배정됩니다.
- 옵션이 없는 상품(단일 SKU)은 이 파일에 등록하지 않아도 됩니다 (자동으로 `MF3` / `FREE` 처리).

## 출력 파일 상세

### 시스템 업로드용 (2개 시트)

**시트 1: 매입상품 구매오더 업로드 양식**
- UID, 스타일번호, 오더 총 수량, 계획 정상가/원가, 사이즈유형이 자동 입력됩니다.
- 옵션 상품은 Size01, Size02 등에 옵션별 수량이 배분됩니다.
- 납품일, 전자결재 등 나머지 컬럼은 직접 입력이 필요합니다.

**시트 2: Barcode 업로드(옵션)**
- 스타일번호, UID, 사이즈(옵션명), 바코드가 자동 입력됩니다.
- 옵션 없는 상품의 사이즈는 "FREE"로 표시됩니다.

### 브랜드 송부용 (1개 시트)

- 브랜드명, 88코드, 상품번호, 상품명, 가격 정보, 총 발주금액, 발주수량이 자동 입력됩니다.
- 상단에 TOTAL 합계 행이 포함됩니다.
- 지정입고일은 직접 입력이 필요합니다.

## 기능 B: 발주 파일 통합

프로그램의 **"B. 발주 파일 통합"** 탭에서 사용합니다.

### 용도
날짜별·브랜드별로 분산된 발주 파일을 하나의 통합 파일로 합칩니다.

### 사용 순서
1. "+ 파일 추가"로 통합할 발주 파일들을 선택 (복수 가능)
   - 드래그앤드롭으로도 파일을 추가할 수 있습니다 (tkinterdnd2 설치 시)
   - 파일 위에 마우스를 올리면 전체 경로가 툴팁으로 표시됩니다.
2. (선택) 상품코드-바코드 매칭 파일을 업로드하면 `상품번호` 자동 채움
3. "통합 실행" 클릭 → `통합_발주수량.xlsx` 생성
   - 이미 같은 이름의 파일이 있으면 덮어쓰기 확인이 표시됩니다.
4. 처리 중 "취소" 버튼으로 중단할 수 있습니다.

### 발주 파일 필수 컬럼

| 컬럼명 | 설명 | 예시 |
|--------|------|------|
| `브랜드명` | 브랜드 이름 | 리브러쉬 |
| `88코드` | 바코드 | 8809915250003 |
| `상품명` | 상품 이름 | 100배 농축 가글 50ml |
| `공급가(VAT 제외)` | 공급 단가 | 8386 |
| `정상가(VAT 포함)` | 정상 판매가 | 25900 |
| `상시행사가(VAT 포함)` | 상시 행사가 | 20500 |
| `발주수량(오프)` | 오프라인 발주 수량 | 100 |

- 파일 상단에 `지정입고일` 정보가 있으면 자동 추출됩니다 (예: "4/1(수)" → 2026-04-01)

### 매칭 파일 필수 컬럼 (선택 업로드)

| 컬럼명 | 설명 |
|--------|------|
| `바코드` | 88코드와 동일한 값 |
| `신규 상품코드` 또는 `1P 상품코드` 또는 `상품코드` | 상품번호 (UID) |

### 출력 컬럼
`지정입고일`, `브랜드명`, `88코드`, `상품번호`, `상품명`, `공급가(VAT 제외)`, `정상가(VAT 포함)`, `상시행사가(VAT 포함)`, `총 발주금액(VAT 제외)` (자동 계산), `발주수량(오프)`

## 작업 이력

프로그램의 **"작업 이력"** 탭에서 과거 변환 기록을 확인할 수 있습니다.

- 기록 항목: 시간, 유형(매칭 변환/파일 통합), 브랜드, 성공/실패 수, 경고 수
- "새로고침" 버튼으로 최신 상태를 불러옵니다.
- "이력 삭제" 버튼으로 전체 이력을 삭제할 수 있습니다.
- 이력은 `~/.mss_converter/history.json`에 저장됩니다 (최대 100건).

---

## 세션 자동 복원

앱을 종료하고 다시 실행하면 마지막 상태가 자동으로 복원됩니다:

- 파일 경로 (파일이 여전히 존재하는 경우에만)
- 저장 위치
- 브랜드 선택 상태
- 마지막 활성 탭

세션 정보는 `~/.mss_converter/session.json`에 저장됩니다.

## 주의사항

1. **파일 형식**: 반드시 `.xlsx` 또는 `.csv` 파일을 사용하세요.
2. **컬럼명 일치**: 입력 파일의 컬럼명이 기존 양식과 동일해야 합니다. 컬럼명을 임의로 변경하면 오류가 발생할 수 있습니다.
3. **동시 실행 금지**: 변환 중에 "변환 실행" 버튼을 다시 누르지 마세요.
4. **출력 파일 덮어쓰기**: 같은 브랜드로 다시 변환하면 확인 다이얼로그가 표시됩니다. "확인"을 누르면 기존 파일이 덮어씌워집니다.
5. **옵션 매핑 경고**: 로그에 경고가 표시되면 "매칭 실패 리포트" 버튼으로 상세 내역을 Excel로 저장하여 확인하세요.
6. **에러 메시지**: 오류 발생 시 한국어로 된 안내 메시지가 표시됩니다. 메시지의 "확인사항"을 참고하여 문제를 해결하세요.

## 문제 해결

| 증상 | 해결 방법 |
|------|----------|
| 파일 선택 시 `[!!]` 표시 | 올바른 파일인지 확인. 필수 컬럼이 포함되어야 합니다. |
| "파일 접근이 거부되었습니다" | Excel에서 파일을 닫고 다시 시도하세요. |
| "브랜드명 컬럼이 없습니다" | 파일 1이 총 발주 수량 데이터셋이 맞는지 확인하세요. |
| 프로그램이 느리게 실행 | 파일 크기가 클 수 있습니다. 불필요한 행/열을 정리해보세요. |
| 프리셋 불러오기 실패 | 저장된 파일이 이동/삭제되었을 수 있습니다. 파일을 다시 선택하세요. |
